## Installing PuzzleJs

To install PuzzleJs with Yarn or Npm, simply:

```bash
yarn add puzzle-microfrontends
```

```bash
npm install --save puzzle-microfrontends
```

Then you can start using it

```js
const PuzzleJs = require('puzzle-microfrontends');
```
