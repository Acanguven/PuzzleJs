## Installing PuzzleJs

To install PuzzleJs with Yarn or Npm, simply:

```bash
yarn add @trendyol/puzzlejs
```

```bash
npm install --save @trendyol/puzzlejs
```

Then you can start using it

```js
const PuzzleJs = require('@trendyol/puzzlejs');
```
